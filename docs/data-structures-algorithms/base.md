## 数据结构

### 线性表

线性表（linear list）是数据结构的一种，一个线性表是 n 个具有相同特性的数据元素的有限序列。
线性表主要由顺序表示或链式表示。在实际应用中，常以栈、队列、字符串等特殊形式使用。
